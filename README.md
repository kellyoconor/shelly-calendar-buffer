# Shelly OAuth Bootstrap

## Setup

1. Unzip this folder
2. Run:
   npm install
3. Set your credentials:
   export GOOGLE_CLIENT_ID="your_client_id"
   export GOOGLE_CLIENT_SECRET="your_client_secret"
4. Run:
   node auth.js

It will open a browser. Log in as shellykelly19103@gmail.com and approve access.
Your refresh token will print in the terminal.
